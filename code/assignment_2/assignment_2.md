# 作业二：基于 MLP 的手写数字识别

## 一、作业背景

多层感知机（MLP, Multi-Layer Perceptron）是最基础的前馈神经网络结构，也是理解深度学习的重要起点。MNIST 数据集包含 70,000 张 28×28 像素的手写数字灰度图像（60,000 张训练集 + 10,000 张测试集），是深度学习领域最经典的基准数据集之一。

本次作业要求同学们使用 PyTorch 框架，在 MNIST 数据集上搭建并训练一个 MLP 模型，实现手写数字（0-9）的分类识别，并开发一个 GUI 应用程序，允许用户手写数字并实时调用训练好的模型进行识别。

## 二、作业内容

本次作业分为两个任务：

### 任务一：模型训练（train.py）

编写 Python 脚本，完成以下功能：

1. **数据准备**：自动下载 MNIST 数据集，并进行预处理（转为 Tensor、标准化）。
2. **模型搭建**：定义一个至少包含 **2 个隐藏层** 的 MLP 模型，使用 ReLU 作为激活函数。
3. **模型训练**：使用交叉熵损失函数和 Adam 优化器进行训练，支持 GPU 加速。
4. **模型保存**：训练完成后将模型参数保存为 `mnist_mlp.pth` 文件。
5. **可视化**：绘制训练损失曲线和测试集准确率曲线。

### 任务二：GUI 手写识别（gui.py）

编写 Python 脚本，完成以下功能：

1. **手写画布**：提供一个可用鼠标手写数字的画布窗口。
2. **模型加载**：加载任务一中训练好的模型文件 `mnist_mlp.pth`。
3. **识别功能**：将手写内容预处理后送入模型，输出预测的数字结果。
4. **置信度展示**：以柱状图形式展示模型对 0-9 各数字的预测置信度。
5. **清除功能**：提供清除画布的按钮，方便多次测试。

## 四、运行方式

```bash

python train.py

# 任务二：启动 GUI 识别
python gui.py
```

## 五、检查方法

1. **训练流程检查**：运行 `train.py`，观察训练过程是否正常，最终测试集准确率是否达到 97% 以上。
2. **模型文件检查**：确认运行后生成了 `mnist_mlp.pth` 模型文件。
3. **训练过程的Loss曲线和正确率曲线**：截图保存。
4. **GUI 功能检查**：运行 `gui.py`，现场在画布上手写数字（0-9 各写一次），检验识别准确性，截图保存。
